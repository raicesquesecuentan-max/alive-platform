## 🎬 GUÍA DE USO - TIPS PARA MEJORES RESULTADOS

### Para **Midjourney**:
- Usa `--v 6` para mejor realismo
- Usa `--q 2` para máxima calidad (consume más créditos)
- Usa `--ar 16:9` para formato cine
- Agregar `--style raw` si quieres menos procesamiento artístico

### Para **DALL-E 3**:
- Copia el prompt completo tal cual
- Especifica "ultra-realistic photography" explícitamente
- DALL-E es excelente para contextos de uso real
- Mejor para renders de personas y escenarios

### Para **Leonardo AI**:
- Excelente para renders técnicos y explosionados
- Usa modelo "Leonardo Diffusion XL" para realismo
- Soporta mejor las especificaciones técnicas detalladas
